# Submission Checklist

## Required

- [ ] Public GitHub repo
- [ ] README with product intro, architecture, deployment info, integrations, team, and X Layer positioning
- [ ] Agent wallet identity documented in README
- [ ] X Layer deployment addresses
- [ ] Onchain OS or Uniswap integration clearly shown
- [ ] Google Form submission before deadline

## Strongly recommended

- [ ] Short demo video
- [ ] Screenshots or GIFs from the prototype
- [ ] Contract addresses and proof verifier transaction examples
- [ ] A simple system diagram in README

## Useful commands

- [ ] Run `pnpm build`
- [ ] Run `pnpm contracts:compile`
- [ ] Run `pnpm submission:bundle`
- [ ] Run `pnpm submission:check`

## Before final submission

- [ ] Replace placeholder proof verification logic
- [ ] Add real contract ABIs or deployment artifacts
- [ ] Add environment setup notes
- [ ] Add actual transaction hashes from X Layer mainnet or test execution
